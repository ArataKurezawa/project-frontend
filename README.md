## Стек и инструменты

- **Vite** — быстрый сборщик и dev-сервер
- **Handlebars** — компонентный шаблонизатор
- **SCSS** — препроцессор стилей с переменными и миксинами
- **ES6+** — модульный JavaScript
- **Flexbox & Grid** — для адаптивной сетки
- **BEM** — методология именования классов
- **ESLint, Prettier** — контроль качества кода

## Установка и запуск

### 1. Установить зависимости

npm install

### 2. Запуск в режиме разработки

npm run dev

Откройте [http://localhost:5173](http://localhost:5173) в браузере.

### 3. Сборка для продакшена

npm run build

Готовые файлы появятся в папке `/dist`.

### 4. Предпросмотр собранного проекта

npm run preview

## Кратко о реализации

- **Адаптивная сетка:** 12/8/4 колонки, резиновая ширина, gap по макету.
- **Типографика:** размеры и line-height строго по макету, на мобильных переопределяются.
- **UI-компоненты:** кнопки, поля, select, чекбокс, radio, аккордеон — все состояния реализованы.
- **Интерактив:** первый элемент каждого типа интерактивный, остальные — статичные состояния.
- **Шрифт:** Graphik LCG, подключён локально через SCSS.
- **Данные:** контент, сетка и пользователь берутся из `src/data/*.json`.
- **Весь контент и контакты** на сайте динамически подставляются из user.json.
